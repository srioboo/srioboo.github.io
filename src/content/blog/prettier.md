- [ ] documentar cambio de linea con prettier

```
# instalar con
npm install --global prettier

# o bien
px prettier --end-of-line crlf --write .

# y lanzar
prettier --end-of-line lf --write .


# con git
git config core.autocrlf false
git rm --cached -r .
git reset --hard
```
